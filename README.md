## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/rkymtc/react-producttable

```

Proje dizinine gidin

```bash
  cd react-producttable

```

Gerekli paketleri yükleyin

```bash
  yarn
```

Sunucuyu çalıştırın

```bash
  yarn start
```

  
## Kullanılan Teknolojiler

- React
- Bootstrap
- Ant Design
